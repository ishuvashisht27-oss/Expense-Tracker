# Simple Expense Tracker

A lightweight Command Line Interface (CLI) application built in Python to track your daily expenses. Data is saved locally in a CSV file.

## Features
* Add expenses with a category and amount.
* View your entire expense history.
* Calculate total spending.
* View a summary of costs by category.

## How to Run
1. Make sure you have **Python 3** installed.
2. Download the script file.
3. Run the script in your terminal:
   ```bash
   python expense_tracker.py
   ```

## File Structure
* `expense_tracker.py` - The main Python program.
* `expense.csv` - The local file where data automatically saves.
